# 民法典婚姻家庭编 · 核心条款

> 婚姻家庭编不是冷冰冰的法条，而是两个人从"恋人"变成"法律共同体"那一刻起，所有权利义务的底层规则。领证前花一个下午过一遍，比婚后花几十万打官司便宜得多。

## 一、结婚条件：五条红线

- **自愿**（1046）：男女双方完全自愿，禁止强迫与干涉。这是"包办逼婚"无效的根源。
- **婚龄**（1047）：男不得早于 22 周岁，女不得早于 20 周岁。没到年龄就算办了酒，也不算法律上的婚姻。
- **禁亲**（1048）：直系血亲或三代以内旁系血亲禁止结婚。熟人社会尤其要查清远房表亲。
- **登记**（1049）：必须亲自到婚姻登记机关申请。**完成登记才确立婚姻关系**——只办酒不领证，法律上是同居。
- **落户**（1050）：登记后按约定，女方可入男方家、男方可入女方家。迁不迁户口是约定，不影响婚姻效力。

## 二、无效与可撤销：四种"法律不保护"的婚姻

- **无效**（1051）：重婚、禁婚亲属、未到法定婚龄——自始没有法律约束力。
- **胁迫可撤销**（1052）：受胁迫结婚，可在胁迫终止之日起一年内申请撤销。
- **隐瞒重大疾病可撤销**（1053）：一方患重大疾病，应在登记前如实告知；不如实告知，对方可在知道起一年内申请撤销，且**撤销自始无效**。这是 2021 年新增的告知义务。
- **后果**（1054）：无效或被撤销的婚姻，当事人不具有夫妻权利义务；同居财产协议优先，协议不成由法院照顾无过错方判决。

> 提醒：婚前体检不是走过场。不如实告知重大疾病，对方一年内可申请撤销，等于这婚没结过。

## 三、夫妻财产制：四个易混点

1. **共同财产（1062）**：婚后所得的工资奖金、生产经营投资收益、知识产权收益、继承或受赠（除非明确只归一方）——归共同所有，平等处理。
2. **个人财产（1063）**：婚前财产、人身损害赔偿、明确只归一方的遗赠、一方专用生活用品。
3. **易混点 · 婚前财产的"收益"**：婚前一套房是个人财产没错，但婚后收的租金、投资收益，可能被认定为共同财产（1062 第二项"投资的收益"）。要约定清楚。
4. **约定财产制（1065）**：双方可书面约定婚前及婚后财产归各自/共同/部分共有。**必须书面**——这是婚前协议、婚内财产约定的法律依据。

## 四、共债共签原则（1064）

- 共同签名或事后追认的债务 = 共同债务。
- 一方以个人名义为**家庭日常生活**所负债务 = 共同债务。
- 一方以个人名义**超出家庭日常生活**所负债务 = **原则上不算共同债务**，除非债主能证明钱花在了家里。

> 这是"共债共签"原则：一方私下借的大额债务，原则上不算共同债务。对备婚情侣是保护，也意味着婚后大额借款最好双方都签字，免得说不清。

## 行动建议

- 领证前两人共读 1046–1065 条，建立"我们是法律共同体"的认知。
- 各列一份财产清单（按 1062/1063 分类）并交换签字确认。
- 把 1053（疾病告知）、1064（共债共签）、1065（约定书面）、1087（分割三原则）抄进手机备忘录。
- 财产结构复杂（股权/继承预期/婚前房贷）的，认真考虑签书面约定，别因"谈钱伤感情"拖着。
