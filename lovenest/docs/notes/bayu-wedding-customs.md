# 巴渝婚俗总览

> 重庆幅员太广，一个"巴渝婚俗"装不下。主城是西式为主、中式点缀；忠县长江边留着汉族老规矩的郑重；彭水苗山深处则是拦门酒、哭嫁、对歌的另一番景象。备婚前先把地图摊开，才知道哪些规矩要守、哪些可以商量。

## 一、源流：长江串起的三次拉扯

巴渝婚俗的根子在长江。古巴国先民沿江而居，婚俗既有中原汉族三书六礼的骨架，又掺进巴人"重鬼神、好巫祝、尚歌舞"的脾性。《华阳国志·巴志》记巴人"质直好义，土风朴厚"——这种气质至今渗在婚礼现场：讲热闹、讲面子、讲直爽，不爱虚头巴脑。

近代以来经历三次拉扯：清末民初西式婚礼随开埠入主城，县乡仍守老礼；建国后移风易俗砍掉大半繁文缛节；2000 年至今传统复兴，彭水作为苗族土家族自治县反而把哭嫁歌、苗绣嫁衣重新摆回婚礼。结果是今天巴渝婚俗呈分层结构。

## 二、三书六礼 → 现代四步

三书六礼是中原汉族经典框架：三书（聘书/礼书/迎书），六礼（纳采/问名/纳吉/纳征/请期/亲迎）。

落到今天的重庆忠县，六礼被压成四步：

1. **提亲**（约等于纳采+问名）：男方长辈带队、带聘礼、有时辰讲究。
2. **订婚**（约等于纳吉+纳征）：定亲、过彩礼。
3. **定日子**（约等于请期）：商定婚期。
4. **接亲**（亲迎）：迎娶新娘过门。

三书基本不写了，但礼金清单、聘礼单子这些"礼书"功能还在，换成红包和烟酒糖肉的实物组合。

## 三、三地差异：四个维度

| 维度 | 主城 | 忠县 | 彭水 |
|---|---|---|---|
| 礼数密度 | 最简，常跳过订婚 | 居中，四步走到 | 最繁，叠加拦门酒/对歌/哭嫁 |
| 彩礼形态 | 象征性或折首付 | 现金+实物，1–6 万常见 3 万 | 区间接近，实物礼（银饰/布匹）更被看重 |
| 婚礼服饰 | 婚纱+西装+敬酒服 | 西式为主，敬茶加秀禾 | 土家西兰卡普 / 苗绣嫁衣配银饰 |
| 宴席 | 酒店圆桌十菜一汤 | 江河鲜、流水席连摆两天 | 长桌宴、坨坨肉、油茶汤、拦门酒 |

## 四、融合主轴：先定骨架再填血肉

融合不是"挑喜欢的各来一点"，要先看清两套体系各自的结构完整性。建议以**汉族巴渝四步为主轴**（男方忠县传统），在每个节点嵌入彭水少数民族元素：

- 提亲：加彭水女方"认亲"环节。
- 订婚：加"换盅"（双方互敬酒定亲）。
- 接亲：嵌入拦门酒、对歌、哭嫁出阁。
- 回门：保持汉族回门礼，礼物按女方家喜好备。

骨架是忠县的，血肉是彭水的，两边都认。

## 避坑

- **别拿主城简化版套忠县**：忠县提亲必须有长辈带队、聘礼、时辰，否则长辈觉得"太随便"。
- **苗 ≠ 土家**：彭水是"苗族土家族自治县"。拦门酒是苗族风俗，哭嫁歌是土家族国家级非遗；苗银与土家西兰卡普是两套独立非遗。女方家到底是哪个民族，必须先问清，写错当场得罪人。
- **非遗不能"砍半"**：哭嫁歌有完整段落，砍到一半比不唱还尴尬。要么做全套，要么坦诚商量用现代简化版。
- **别忘 280 公里**：忠县到彭水自驾四小时，接亲车队、住宿、聘礼运输都要算进路线规划。

## 行动建议

- 跟双方父母各通一次长电话：咱家最看重哪几步？对方家有什么规矩绝对不能省？记进备忘录。
- 确认女方家民族归属（苗族/土家族/混居），决定后续少数民族元素方向。
- 物色一个"两头懂"的媒人/知客，到提亲阶段是关键人物。
